# Birthrates 2025

Births per 1000 population in the United States, starting in 1965. There has been concern that the birthrate may be declining. A good model for tends in birthrate may allow for some prediction.

The data are in `Birthrates_2025.txt`


This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
