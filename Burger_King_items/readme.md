# Burger King items

Burger King publishes full nutrition information on its menu. These data are for the foods on the menu recently. (Visit the site listed as the reference for the most current list.)

The data are in `Burger_King_items.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Item` | 
`Serving.size` | 
`Calories` | 
`Fat.Cal` | 
`Protein.g.` | 
`Fat.g.` | 
`Sat.Fat.g.` | 
`Trans.fat.g.` | 
`Chol.mg.` | 
`Sodium.mg.` | 
`Carbs.g.` | 
`Fiber.g.` | 
`Sugar.g.` | 
`Meat` | 
`Breakfast` | 
`Not.Breakfast` | 
`CarbsxMeat` |
