# Marriage age 2024

The estimated median age at fist marriage by sex from 1890 to 2017 is provided by the U.S. Census bureau. Since 1960, marriage ages have been increasing steadily. Has the difference between men's and women's first marriage age changed?

The data are in `Marriage_age_2024.txt`


This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
