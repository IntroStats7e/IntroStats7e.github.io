# Emails

A university teacher saved every e-mail receive from students in a large introductory statistics class during one term. He then counted, for each student who had sent him at least one e-mail, how many e-mails each student had sent. What is the distribution of e-mail communications?

The data are in `Emails.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Number_of_E.mails` | Number of emails sent by each individual student

This dataset is sourced from: Velleman personal communication
