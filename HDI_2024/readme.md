# HDI 2024

The United Nations Development Programme (UNDP) uses the Human Development Index (HDI) in an attempt to summarize in one number the progress in health, education, and economics of a country. The dataset includes components of the HDI such as life expectancy, shooting, and GNI/capita.

The data are in `HDI_2024.txt`

The variables in this dataset are:
