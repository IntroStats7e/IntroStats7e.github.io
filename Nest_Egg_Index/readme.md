# Nest Egg Index

The Nest Egg Index, devised by the investment firm A.G. Edwards, is a measure of saving and investment performance for each of the 50 states, based on 12 economic factors, including participation in retirement savings plans, personal debt levels, and home ownership. The average index is 100, and the numbers indicate values above or below that average.

The data are in `Nest_Egg_Index.txt`

The variables in this dataset are: 

Header | Description
---|--------
`State` | State
`Nest.Egg.Index` | Index value (percent of the average value of 100)
`Region` | Region of state (South/West or Northeast/Midwest)

This dataset is sourced from: http://www.statemaster.com/graph/eco_nes_egg_ind-economy-nest-egg-index
