# Alcohol Consumption 2024

These are the annual numbers of per capita wine, beer, and spirits consumption (in gallons) in America from 1970 to 2022. The data are primarily derived from state sales and taxation records. 

The data are in `Alcohol_Consumption_2024.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Year` | Year 
`Beer` | Per capita beer consumption in the U.S. (gallons)  
`Spirits` | Per capita spirits consumption in the U.S. (gallons) 
`Wine` | Per capita wine consumption in the U.S. (gallons) 

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
