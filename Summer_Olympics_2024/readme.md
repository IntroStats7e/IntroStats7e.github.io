# Summer Olympics 2024

Sixty three countries won gold medals in the 2024 Summer Olympics. The dataset lists them, along with the total number of gold medals each won. It can be a challenge to find a good display for data like these.

The data are in `Summer_Olympics_2024.txt`

The variables in this dataset are:

`Country` | 
`Gold` | 
`Silver` |

`Total` |
