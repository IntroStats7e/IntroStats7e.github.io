# Scottish Hill Races

Hill races are races that climb generally steep hills, held throughout Scotland throughout the year. The file holds records for men and women in these races the last time those were posted in an accessible table along with facts about the races. In particular, we know the length(km) and total climb(m). These are two independent predictors of the record times. Sex of the runner can be an additional indicator variable.

The data are in `Scottish_Hill_Races.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Race` | 
`Distance.km.` | 
`Climb..m.` | 
`Mens.Time..min.` | 
`Womens.Time..min.` | 

This dataset is sourced from: http://www.scottishhillracing.co.uk
