# COLall 2025

The cost of a variety of common items in 313 cities around the world in $ as of January 2025, adjusted so that New York, U.S.A. is 100.

The data are in `COLall_2025.txt`

The variables in this dataset are: 

Header | Description
---|--------
`City` | 
`Water.1.5.l.` | 
`Eggs.dozen.` | 
`Bread.white.500g.` | 

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
