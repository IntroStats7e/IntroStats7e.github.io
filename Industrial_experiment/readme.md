# Industrial experiment

Engineers at a computer production plant tested two methods for accuracy in drilling holes into a PC board. They tested how fast they could set the drilling machine by running 10 boards at each of two different speeds. To assess the results, they measured the distance (in inches) from the center of a target on the board to the center of the hole.

The data are in `Industrial_experiment.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Distance` | Distance (inches) from target drilling location to hole 
`Speed` | Drilling machine speed setting
