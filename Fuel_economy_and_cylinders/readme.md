# Fuel economy and cylinders

Fuel economy (mpg) and the number of cylinders in a sample of cars. Data extracted from a larger cars dataset.

The data are in `Fuel_economy_and_cylinders.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Fuel.Efficiency..mpg.` | Car's fuel economy (miles per gallon) 
`Cylinders` | Number of cylinders
