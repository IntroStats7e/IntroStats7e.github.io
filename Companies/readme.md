# Companies

Facts about companies selected from the Forbes 500 list for 1986. This is a 1/10 systematic sample from the alphabetical list of companies. The Forbes 500 includes all companies in the top 500 on any of the criteria, and thus has almost 800 companies in the list.

The data are in `Companies.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Company` | Company name 
`Assets` | 
`Sales` | 
`Market.Value` | 
`Profits` | 
`Cash.Flow` | 
`Employees` | Number of employees (thousands) 
`sector` | Business sector
`Banks` | Indicator that company is a bank (1=yes, 0=no)

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
