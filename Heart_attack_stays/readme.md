# Heart attack stays

Number of days spent in hospital by female patients admitted to hospitals in New York during one year with a primary diagnosis of acute myocardial infarction (heart attack). Data are from public Medicare records. The data also include the age of the patient.

The data are in `Heart_attack_stays.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Stay.days.` | Length of hospital stay (days) 
`Age.years.` | Patient age (years) 

This dataset is sourced from: Data originally from Medicare, not citable
