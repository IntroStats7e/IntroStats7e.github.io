# Disk drives 2025

Disk drive capacity is often given in terabytes (TB), where 1 TB = 1000 gigabytes, or about a trillion bytes. A search of prices for external disk drives on Amazon.com in mid-2016 found the data on capacity and price.

The data are in `Disk_drives_2025.txt`


This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
