# Brewpubs 2024

This dataset reports the number of brewpubs in the United States for each year in the past three decades (1994-2024). 

The data are in `Brewpubs_2024.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Brewpubs` |  Number of brewpubs in the United States that year
`Year` | Year

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
