# Kentucky Derby 2025

This folder contains the Kentucky Derby 2025 data. The data are sourced from www.kentuckyderby.ag/kentuckyderby-results.php or https://en.wikipedia.org/wiki/Kentucky_Derby. 

The variables in this dataset are: 

Header | Description
---|---------
`Year` | Calendar year of race
`Year_no` | Years since 1875 
`Winner` | Name of winning horse
`Mins` | Minutes for winning time
`Secs` | Seconds for winning time
`Time.in.Sec` | Winning time (seconds) 
`Distance..mi.` | Race distance (miles) 
`Speed` | Speed (mph) 
`Jockey` | Name of winning horse's jockey
`Trainer` | Name of winning horse's trainer
`Owner` | Name of winning horse's owner
