# Car speeds 100

John Beale of Stanford, California, recorded the speeds of cars driving past his house, where the speed limit read 20 mph (He actually recorded every car for a two-month period. These are 100 representative readings.)

The data are in `Car_speeds_100.txt`

The variables in this dataset are: 

Header | Description
---|--------
`speed.mph.` | Speed of car (miles per hour)

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
