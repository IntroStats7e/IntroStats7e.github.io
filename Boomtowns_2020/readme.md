# Boomtowns 2020

In 2020, the website NewGeography.com listed its ranking of the best cities for job growth in the United States. The magazineâ€™s top 20 large cities, along with their weighted job rating indices, are given in the table. The full data set contains 71 cities.

The data are in `Boomtowns_2020.txt`

The variables in this dataset are: 

Header | Description
---|--------
`City` | City name
`Index` | Job growth index (availability of employment)
`Nonfarm.employment` | Number of non-farm jobs (in thousands) 
`Size` | "Large" areas have a current nonfarm employment base of at least 450,000 jobs. "Midsize" areas range from 150,000 to 450,000 jobs. "Small" areas have as many as 150,000 jobs.

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
