# Fuel efficiency

We know from common sense and from Physics that heavier cars need more fuel, but exactly how does a car’s weight affect its fuel efficiency? The data set continues data on 38 cars including their fuel efficiency in miles per gallon measured on a track.

The data are in `Fuel_efficiency.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Model` | 
`Eng.Size` | 
`Cylinders` | 
`MSRP` | 
`City.Mpg` | 
`Highway.Mpg` | 
`Weight` | 
`Type` | 
`Country` | 

This dataset is sourced from: Consumer Reports
