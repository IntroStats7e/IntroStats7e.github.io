# Smoking by Age and Gender

The Centers for Disease Control and Prevention provide data on smoking rates by year and for men and women separately.

The data are in `Smoking_by_Age_and_Gender.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Age` | 
`Gender` | 
`X1974` | 
`X1994` | 
`X2004` | 
`X2014` | 
`X2024` | 

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
