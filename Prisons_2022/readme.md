# Prisons 2022

This dataset, drawn from the Bureau of Justice Statistics, reports the number of people incarcerated in prisons in each state in 2021. 

The data are in `Prisons_2022.txt`

The variables in this dataset are: 

Header | Description
---|--------
`State` | State 
`Total_2021` | Total number of people incarcerated in that state in 2021
`Men_2021` | Number of men incarcerated in 2021
`Women_2021` | Number of women incarcerated in 2021

This dataset is sourced from: Bureau of Justice Statistics http://www.bjs.gov/ https://bjs.ojp.gov/document/p22st.pdf
