# Bird Species 2013

The Audubon Society holds an annual Christmas Bird Count (www.birdsource.org), in which bird watchers at various locations around the country see how many different species of birds they can spot. We have data on the number of species counted from the 20 sites with the most species in 2013. 

The data are in `Bird_Species_2013.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Counts` | Species count at each site

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
