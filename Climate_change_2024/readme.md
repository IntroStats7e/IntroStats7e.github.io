# Climate change 2024

Changes in the mean global temperature are primarily due to increases in CO2 levels. Both trends are here from 1958 to 2024.

The data are in `Climate_change_2024.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Year` | 
`Temperature.Anomaly..degC.` | 
`CO2.Concentration..ppm.` | 

This dataset is sourced from: Global temperature from https://data.giss.nasa.gov/gistemp/graphs/graph_data/Global_Mean_Estimates_based_on_Land_and_Ocean_Data/graph.txt  CO2 from https://scrippsco2.ucsd.edu/assets/data/atmospheric/stations/in_situ_co2/monthly/monthly_in_situ_co2_mlo.csv
