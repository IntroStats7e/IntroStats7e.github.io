# Wisconsin ACT 2023

The data give the mean ACT composite scores for all Wisconsin public schools in 2022-2023 along with the type of school and number of students. How do charter schools compare with public schools.

The data are in `Wisconsin_ACT_2023.txt`


This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
