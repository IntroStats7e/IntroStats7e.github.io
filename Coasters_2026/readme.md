# Coasters 2026

The website rcdb.com, the Roller Coaster Database, holds facts about every roller coaster in the world, current or past. (If you know of one that is missing, please let the site master know.) These data are for coasters in operation as of January 2026.  

The data are in `Coasters_2026.txt`

The variables in this dataset are: 

Header | Description
---|---------
`X` | 
`Roller.Coaster` | 
`Amusement.Park` | 
`Angle` | 
`Inversions` | 
`Duration` | 
`track_type` | 
`speed_mph` | 
`height_ft` | 
`drop_ft` | 
`length_ft` | 

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
