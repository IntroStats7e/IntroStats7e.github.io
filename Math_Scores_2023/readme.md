# Math Scores 2023

The National Center for Education Statistic reports average mathematics achievement scores for eighth  graders in all 50 states.

The data are in `Math_Scores_2023.txt`

The variables in this dataset are: 

Header | Description
---|--------
`State` | 
`Score` | 
`Change.from.2019` | 

This dataset is sourced from: https://www.niaaa.nih.gov/sites/default/files/pcyr1970-2022.txt
