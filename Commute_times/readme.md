# Commute times

Commute lengths are reported for workers at a particular company. The data have been modified to protect confidentiality.

The data are in `Commute_times.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Commute.Time` | Length of commute (minutes)

This dataset is sourced from: Data from a survey of workers at a company. The data have been modified to protect confidentiality.
