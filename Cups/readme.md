# Cups

For her class project, a student compared the effectiveness of various coffee cups. She compared four different cups, testing each of them eight different times. Each time, she heated water to 180 degrees Fahrenheit, poured it into a cup, and sealed it. After 30 minutes, she measured the temperature again and recorded the difference in temperature. Because these are temperature differences, smaller differences mean that the liquid stayed hot -- just what we would want in a coffee cup. 

The data are in `Cups.txt`

The variables in this dataset are: 

Header | Description
---|--------
`Difference` | Temperature difference after 30 minutes (degrees F) 
`Container` | Cup used 

This dataset is sourced from: From Stat 231 De Veaux’s class project
